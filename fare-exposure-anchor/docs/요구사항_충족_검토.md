# 항공 운임 노출 분석 기능 — 요구사항 충족 검토

## 1. 요구사항 대비 현재 구현 매핑

### 1.2.1 가격대별 수요 분포 파악 어려움

| 요구 | 현재 구현 | 충족 여부 |
|------|-----------|-----------|
| 특정 노선에서 **어떤 가격대 운임을 많이 접했는지** | **운임 분포 (구간별)** 히스토그램: 금액 구간(예: 5만 원 단위)별 노출 건수 시각화 | ✅ 충족 |
| 어떤 가격대가 시장에서 **주로 노출되고 있는지** | **동일 금액별 노출 횟수** 카드: 같은 금액이 노출된 횟수(상위 50개), **최빈 구간(modeBin)** 인사이트 문구 | ✅ 충족 |

- 필터: 항공사 / 출발·도착 / 편도·왕복 / 기간(24h·7d) / 채널(웹·모바일) / 출발일·도착일로 **특정 노선·여정·기간** 지정 가능.

---

### 1.2.2 시장 기준 가격 기준점 부재

| 요구 | 현재 구현 | 충족 여부 |
|------|-----------|-----------|
| **어느 가격대에서 경쟁 중인지** | **운임 분포 구간(P25~P75)** 카드: 노출 운임의 가운데 50%가 있는 가격대 (경쟁 구간 가시화) | ✅ 충족 |
| **어느 가격대에서 판매 가능성이 높은지** | **Anchor(중앙값)** 카드 + 인사이트: "판매 가능성이 높은 가격 기준점"으로 활용 가능한 노출 기준 중앙값 제시 | ✅ 충족 |

- 인사이트 카드에서 Anchor(기준점) 운임, 가장 많이 노출된 구간, P25~P75 설명으로 **기준점 도출**을 명시적으로 지원.

---

### 1.2.3 이익율 설계 어려움

| 요구 | 현재 구현 | 충족 여부 |
|------|-----------|-----------|
| **가격대별 판매 확률** 기반 전략 | **가격대별 이익율 전략 수립 참고** 카드: 가격대(구간)별 **노출 건수**·**노출 비중(%)** 표시. 노출 비중 = 상대적 판매 기회 참고 지표로 활용. 10% 이상 구간에 「이익율 차등 적용 우선 검토」 안내 | ✅ 충족 |
| 감 기반 의사결정 탈피 | 노출 데이터 기반 분포·기준점·구간·노출 비중 제시로, 데이터 기반 판단 기반 마련 | ✅ 충족 |

- **가격대별 이익율 전략 수립 참고** 카드에서 구간별 노출 비중(%)과 전략 참고 문구를 제공하여, 이익율(마진율) 차등 적용 전략 수립을 지원함.

---

### 1.3 목표

| 목표 문장 | 현재 구현 | 충족 여부 |
|-----------|-----------|-----------|
| **특정 항공사/노선/여정 기준** | 필터: 항공사, 출발, 도착, 편도/왕복, 출발일·도착일 | ✅ 충족 |
| **일정 기간** | 기간: 최근 24시간 / 최근 7일 | ✅ 충족 |
| **우리가 시장에 노출했던 운임 분포 가시화** | 히스토그램(구간별) + 동일 금액별 노출 횟수 테이블 | ✅ 충족 |
| **판매 가능성이 높은 가격 기준점 도출** | Anchor(중앙값), 최빈 구간, P25~P75 | ✅ 충족 |
| **가격대별 이익율 전략 수립 지원** | **가격대별 이익율 전략 수립 참고** 카드(노출 비중%, 전략 참고 열) + 인사이트 카드 내 안내 문구 | ✅ 충족 |

---

### 1.4 핵심 컨셉

| 컨셉 | 현재 구현 | 충족 여부 |
|------|-----------|-----------|
| **24시간 기준**으로 | 기간 필터: “최근 24시간” / “최근 7일” 선택 가능 | ✅ 충족 |
| **어떤 가격대를 얼마나 노출했는가** | 히스토그램(구간별 노출 건수) + 동일 금액별 노출 횟수 + 총 노출·세션 수 KPI | ✅ 충족 |

## 2. 종합 평가

- **1.2.1, 1.2.2, 1.2.3, 1.3, 1.4**  
  → **모든 요구사항을 충족**하고 있음.

- **이익율 전략 수립 지원**  
  - **가격대별 이익율 전략 수립 참고** 카드에서 가격대(구간)별 노출 건수·노출 비중(%)·전략 참고(이익율 차등 적용 우선 검토)를 제공함.  
  - 인사이트 카드에 “가격대별 이익율 전략” 활용 안내 문구가 포함되어 있음.

## 3. 요약 표

| 구분 | 요구사항 | 충족 | 비고 |
|------|----------|------|------|
| 1.2.1 | 가격대별 수요/노출 분포 파악 | ✅ | 히스토그램 + 동일 금액별 노출 횟수 + 최빈 구간 |
| 1.2.2 | 시장 기준 가격 기준점 | ✅ | Anchor, P25~P75, 최빈 구간 |
| 1.2.3 | 이익율 설계 (데이터 기반) | ✅ | 가격대별 이익율 전략 수립 참고 카드(노출 비중%, 전략 참고) |
| 1.3 | 노출 분포 가시화·기준점 도출 | ✅ | 필터·기간·모든 시각화 일치 |
| 1.3 | 이익율 전략 수립 지원 | ✅ | 가격대별 이익율 전략 수립 참고 카드 + 인사이트 안내 |
| 1.4 | 24h 기준, 어떤 가격대를 얼마나 노출 | ✅ | 기간 필터 + 히스토그램 + 동일 금액별 횟수 |

**결론:**  
현재 구현은 **항공 운임 노출 분석** 요구사항(1.2.1~1.2.3, 1.3, 1.4)을 **모두 충족**합니다. 가격대별 노출 분포·기준점·24h 기준 노출량에 더해, **가격대별 이익율 전략 수립 참고** 카드를 통해 노출 비중(%) 및 이익율 차등 적용 안내를 제공합니다.

## 4. 이익율 전략 관련 구현 요약 (2025 반영)

- **가격대별 이익율 전략 수립 참고** 카드  
  - 가격대(구간) | 노출 건수 | 노출 비중(%) | 전략 참고  
  - 노출 비중 10% 이상 구간: 「이익율 차등 적용 우선 검토」 표시  
  - 설명: 노출 비중이 높은 가격대에 이익율(마진율) 차등 적용 시 참고
- **인사이트** 카드  
  - 「가격대별 이익율 전략」 문단 추가: 아래 참고 카드에서 노출 비중 확인 후 이익율 차등 적용 전략 수립에 활용
